# CSC 161 Spring 2026

This repository contains my work for **CSC 161: Imperative Problem Solving** in Spring 2026. The course focuses on problem solving in **C**, with an emphasis on core imperative programming ideas such as variables, functions, loops, conditionals, arrays, strings, pointers, and program design.

This repo is being used to organize homework and course projects completed during the semester.

## Current Projects

- `Yahtzee game` - a simplified single-player Yahtzee game written in C
- `Connect Four game` - a two-player Connect Four game written in C

## About The Course

CSC 161 introduces imperative programming through the C language. Because C is close to the hardware, the course emphasizes careful thinking about program structure, memory, and efficiency while building a strong foundation for later computer science courses.
